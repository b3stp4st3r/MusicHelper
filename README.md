# Music Helper

Маленькая и легкая утилита для Windows, которая позволяет управлять музыкой с помощью горячих клавиш, даже если плеер свернут или вы находитесь в другой программе.

## Зачем это нужно?
Многие плееры не имеют горячих клавиш для переключения/паузы треков. Лично я столкнулся с этим в Яндекс Музыке и решил создать данный проект.

## Горячие клавиши
* `Ctrl + Alt + K` — Следующий трек
* `Ctrl + Alt + L` — Предыдущий трек
* `Ctrl + Alt + O` — Пауза / Воспроизведение

## Особенности
- **Автозагрузка**: При первом запуске прописывает себя в реестр (`HKCU\...\Run`), чтобы стартовать вместе с Windows.
- **Трей**: Иконка живет в системном трее. Нажмите правой кнопкой мыши, чтобы выйти.
- **Легкость**: Написано на чистом C++ и WinAPI, потребляет минимум ресурсов (практически 0% процессора и пару мегабайт ОЗУ).

## Как собрать
Проект написан на C++ и использует стандартные библиотеки Windows. Вы можете скомпилировать его в Visual Studio или через MinGW:
```bash
g++ main.cpp -o MusicHelper.exe -luser32 -lshell32 -ladvapi32
```
